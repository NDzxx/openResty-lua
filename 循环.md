#循环

```lua
while(condition)

```
##for
* 数值for循环  
```lua
  for var=exp1,exp2,exp3 do  
    <执行体>  
end  
```
var从exp1变化到exp2，每次变化以exp3为步长递增var，并执行一次"执行体"。exp3是可选的，如果不指定，默认为1。

for的三个表达式在循环开始前一次性求值，以后不再进行求值。 
例子：
```lua
function f(x) 
    print("function") 
    return x*2 
end 
for i=1,f(5) do print(i) 
end

```
