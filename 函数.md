#函数
例子：

```lua
--[[ 函数返回两个值的最大值 --]] 
function max(num1, num2) //如果你需要设置函数为局部函数需要使用关键字 local。
    if (num1 > num2) 
    then 
        result = num1; 
    else 
        result = num2; 
    end 

    return result; //Lua语言函数可以返回多个值，每个值以逗号隔开。
end 
-- 调用函数 
print("两值比较最大值为 ",max(10,4))
print("两值比较最大值为 ",max(5,6))

```
